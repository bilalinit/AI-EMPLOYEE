# Weekly CEO Briefing

## Description
Generate a comprehensive weekly business audit report for the CEO. Reads from Odoo accounting system and AI Employee logs to provide business insights.

## Instructions

### 1. Gather Data Sources

#### From Odoo Accounting
Use these MCP tools to gather financial data:

**Revenue Analysis:**
```
mcp__odoo__get_revenue(period="this_week")
mcp__odoo__get_revenue(period="this_month")
```

**Expense Analysis:**
```
mcp__odoo__get_expenses(period="this_week")
mcp__odoo__get_expenses(period="this_month")
```

**Recent Invoices:**
```
mcp__odoo__get_invoices(limit=20)
```

**Recent Payments:**
```
mcp__odoo__get_payments(limit=20)
```

#### From Business Goals
Read: `AI_Employee_Vault/Business_Goals.md`
- Check revenue targets
- Check active projects
- Check key metrics

#### From Activity Logs
Read: `AI_Employee_Vault/Logs/` for recent activity
Read: `AI_Employee_Vault/Done/` for completed tasks

#### From Social Media (Content Posted)
Read: `AI_Employee_Vault/Content_To_Post/posted/` for this week's posts
- Count LinkedIn posts made (LINKEDIN_POST_*.md with status: posted)
- Count Twitter posts made (TWITTER_POST_*.md with status: posted)
- Count Meta posts made (META_POST_*.md with status: posted)
- Check post dates from frontmatter `posted:` field

#### From Email Activity
Read: `AI_Employee_Vault/Logs/` for email-related logs
- Check for EMAIL_SENT_*.md logs to count emails sent this week
- Note any important email threads logged

### 2. Analyze the Data

#### Revenue Analysis
- This week's revenue
- Month-to-date revenue vs target
- Trend compared to previous weeks

#### Expense Analysis
- This week's expenses
- Any unusual or large expenses
- Subscription costs to review

#### Task & Project Status
- Tasks completed this week
- Pending tasks
- Bottlenecks identified

### 4. Email the Briefing to CEO

After generating the briefing, send it via email:

**Read the generated briefing file** and create an email:

Use: `mcp__ai-gmail__send_email`

Parameters:
- `to`: [CEO's email address - read from Business_Goals.md or ask]
- `subject`: "CEO Briefing - Week of [start_date] to [end_date]"
- `body`: [Briefing content as plain text - convert markdown to readable format]

**Email body format** (simplified for email):
```
CEO BRIEFING - Week of [date range]
=====================================

EXECUTIVE SUMMARY
[Summary text]

REVENUE PERFORMANCE
This Week: $[amount]
Month to Date: $[amount] / [target] ([percent]%)
Trend: [up/down/flat]

EXPENSES
This Week: $[amount]
Net Profit: $[amount]

SOCIAL MEDIA
LinkedIn: [count] posts
Twitter: [count] posts
Facebook/IG: [count] posts

EMAIL ACTIVITY
Emails Sent: [count]

PENDING TASKS
- [Task list]

Full briefing attached in AI_Employee_Vault/Briefings/
```

### 5. Update Dashboard
Add briefing summary to `AI_Employee_Vault/Dashboard.md`:
```markdown
### Latest Briefing
- [Date]: Weekly CEO Briefing available - [Briefings/YYYY-MM-DD_Weekly_CEO_Briefing.md]
```

## MCP Tools Available
- **mcp__odoo__get_revenue** - Get revenue for period
- **mcp__odoo__get_expenses** - Get expenses for period
- **mcp__odoo__get_invoices** - Get recent invoices
- **mcp__odoo__get_payments** - Get recent payments
- **mcp__odoo__get_odoo_status** - Get system status
- **mcp__ai-gmail__send_email** - Email briefing to CEO

## Notes
- Run this skill weekly (recommended: Sunday night or Monday morning)
- Compare against targets in Business_Goals.md
- Flag anything needing human attention
- Be proactive with suggestions - don't just report data
- Briefing is both saved to Briefings/ AND emailed to CEO
- Check Content_To_Post/posted/ for social media stats (filter by `posted:` date within this week)
